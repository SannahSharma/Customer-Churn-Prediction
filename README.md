# Customer Churn Prediction Model

Machine learning project to predict telecom customer churn using Python, Scikit-learn and Power BI.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Power BI

## Dataset
IBM Telco Customer Churn Dataset

- 7043 customers
- 21 features

## Models Used

Logistic Regression  
Accuracy: 82.19%

Random Forest  
Accuracy: 78.92%

## Key Churn Drivers

- Total Charges
- Tenure
- Monthly Charges
- Internet Service
- Payment Method

## Dashboard

Power BI dashboard showing:
- Churn Rate
- Customer Risk Levels
- High Risk Customers
- Churn Probability
